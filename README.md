# Lost and Found AI

## Problem Statement
Lost items such as ID cards, books, and personal belongings are common on campus. Existing solutions rely on unorganized WhatsApp groups, making recovery inefficient.

## Solution
Lost and Found AI is an AI-powered campus platform that allows users to upload lost and found items using images or descriptions. The system intelligently matches items and notifies users when a potential match is found.

## Google Technologies Used
- Gemini Vision API
- Firebase Authentication
- Firebase Firestore
- Firebase Cloud Storage

## Features
- Campus-restricted login
- Upload lost and found items
- AI-based image and text matching
- Real-time notifications
- Secure and centralized system

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)

## MVP
The MVP demonstrates the core functionality of uploading items and AI-based matching through a working prototype and demo video.
